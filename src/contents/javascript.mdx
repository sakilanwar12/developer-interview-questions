### 1. What is JavaScript?

JavaScript is a high-level, versatile programming language primarily used to create interactive and dynamic web applications.
It can run both in the browser and on the server (using environments like Node.js).

---

### 2. What is hoisting?

Hoisting is JavaScript's default behavior of moving declarations to the top of the scope.
